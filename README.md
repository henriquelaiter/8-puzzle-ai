# 8 Puzzle Solver in C

Projeto desenvolvido em C para resolução do clássico problema do 8 Puzzle utilizando algoritmos de busca em Inteligência Artificial.

---

## 📚 Sobre o Projeto

O 8 Puzzle é um quebra-cabeça composto por uma matriz 3x3 contendo peças numeradas e um espaço vazio. O objetivo é reorganizar as peças até atingir o estado final desejado.

O sistema permite:
- Solução manual;
- Solução automática;
- Exibição do caminho percorrido;
- Geração de tabuleiros válidos aleatórios.

---

## 🚀 Algoritmos Implementados

- Breadth-First Search (BFS)
- Iterative Deepening Depth-First Search (IDDFS)

---

## 🛠️ Tecnologias Utilizadas

- Linguagem C

- Pilhas

## 📌 Funcionalidades

- Verificação de solvabilidade;
